# Zaymmo Storytelling

## PRINCIPE

Une annonce Zaymmo ne décrit pas un bien — elle raconte une vie.
L'acheteur doit se projeter, ressentir, imaginer sa vie dans ce bien.

```
Description technique → MAUVAIS
"Appartement 85m², 3 pièces, cuisine équipée, DPE B"

Storytelling → BON
"Imaginez vos matins baignés de lumière dans ce 85m² conçu pour la vie moderne.
 La cuisine ouverte invite aux échanges, tandis que les 3 pièces lumineuses
 offrent à chacun son espace. DPE B — votre confort, sans compromis."
```

---

## STRUCTURE NARRATIVE EN 4 ACTES

### Acte 1 — L'invitation (premier paragraphe)
```
Objectif : Accrocher en 2 phrases — donner envie de lire la suite
Technique : Commencer par une image, une sensation, un bénéfice
Éviter    : Commencer par "Nous vous proposons" ou "À vendre"

Formules d'accroche :
"Bienvenue dans [TYPE] où [BÉNÉFICE PRINCIPAL]..."
"Rare sur le marché — [TYPE] [SURFACE] [LOCALISATION] [ATOUT UNIQUE]..."
"[VILLE] offre ses plus belles adresses — en voici l'une..."
"Imaginez [SCÈNE DE VIE] dans ce [TYPE] d'exception..."
"[ADJECTIF FORT] [TYPE] qui réconcilie [VALEUR 1] et [VALEUR 2]..."
```

### Acte 2 — L'architecture et la vie (deuxième paragraphe)
```
Objectif : Décrire les espaces en termes de vie, pas de dimensions
Technique : Volumes, lumière, circulation, convivialité
Éviter    : Lister les pièces comme un inventaire

BON   : "L'agencement ouvert favorise une circulation naturelle entre les espaces de vie"
MAUVAIS : "Il y a un salon, une cuisine et 3 chambres"
```

### Acte 3 — Les détails qui font la différence (troisième paragraphe)
```
Objectif : Les équipements et prestations mis en valeur
Technique : Chaque équipement = un bénéfice concret pour l'acheteur
Éviter    : La liste brute d'équipements

BON   : "La terrasse sud-est invite aux dîners al fresco jusqu'en octobre"
MAUVAIS : "Il y a une terrasse de 15m²"
```

### Acte 4 — L'appel à l'action (quatrième paragraphe)
```
Objectif : Donner envie de visiter — créer l'urgence douce
Technique : Synthèse des atouts + invitation personnalisée
Éviter    : "N'hésitez pas à nous contacter"

BON   : "Cette [TYPE] d'exception mérite d'être découverte — [AGENT] vous accueille pour une visite privée."
```

## VOCABULAIRE STORYTELLING PAR TYPE DE BIEN

### Maison / Villa
```
Verbes d'action : s'épanouit, rayonne, invite, révèle, offre, conjugue
Sensations      : chaleur, sérénité, espace, liberté, intimité
Lieux de vie    : havre de paix, écrin de verdure, domaine, demeure
Moments         : dimanches en famille, soirées d'été, matins ensoleillés
```

### Appartement
```
Verbes d'action : s'ouvre, baigne, dévoile, surprend, optimise
Sensations      : luminosité, fluidité, modernité, élégance, praticité
Lieux de vie    : pied-à-terre idéal, nid douillet, espace de vie
Moments         : retours du travail, réceptions entre amis, télétravail
```

### Loft / Atypique
```
Verbes d'action : transcende, réinvente, inspire, libère, étonne
Sensations      : volume, singularité, caractère, créativité, urbanité
Lieux de vie    : espace unique, loft d'architecte, bien hors du commun
Moments         : projets créatifs, lifestyle urbain, collections d'art
```

### Prestige
```
Verbes d'action : incarne, sublima, distingue, consacre, perpétue
Sensations      : excellence, raffinement, prestige, noblesse, harmonie
Lieux de vie    : propriété d'exception, demeure de prestige, bijou
Moments         : réceptions, héritages familiaux, investissements pérennes
```

## ADAPTATION ÉMOTIONNELLE PAR PROFIL ACHETEUR

### Famille avec enfants
```
Mettre en avant : espace, sécurité, jardin, proximité écoles
Ton             : chaleureux, rassurant, pratique
Arguments       : "les enfants s'épanouissent", "idéal pour grandir ensemble"
```

### Couple / Premier achat
```
Mettre en avant : modernité, DPE, charges maîtrisées, potentiel
Ton             : enthousiaste, tourné vers l'avenir
Arguments       : "premier investissement idéal", "votre chez-vous enfin"
```

### Investisseur
```
Mettre en avant : rentabilité, DPE, localisation, charges
Ton             : factuel, chiffré, rassurant
Arguments       : "rendement locatif optimisé", "valeur patrimoniale"
```

### Retraités / Seniors
```
Mettre en avant : plain-pied, ascenseur, calme, accessibilité
Ton             : serein, valorisant, pratique
Arguments       : "vie simplifiée", "sérénité au quotidien"
```

## RÈGLES ANTI-CLICHÉS

```
INTERDITS — expressions à bannir :
❌ "coup de cœur assuré"
❌ "ne pas manquer"
❌ "opportunité rare" (sauf si vraiment rare)
❌ "bien d'exception" (sauf prestige avéré)
❌ "charmant" (trop vague)
❌ "lumineux" seul (préciser : "baigné de lumière naturelle tout au long de la journée")
❌ "fonctionnel" (trop technique)
❌ "agréable" (trop vague)
❌ "beau" (montrer, ne pas dire)
❌ "prestations de qualité" (préciser lesquelles)

REMPLACEMENTS :
"lumineux" → "baigné de lumière naturelle grâce à [ORIENTATION]"
"beau jardin" → "jardin paysagé de [X]m² sans vis-à-vis"
"belle cuisine" → "cuisine entièrement équipée avec [DÉTAILS]"
"bon état" → "entretenu avec soin / rafraîchi en [ANNÉE]"
```

## FORMULES POUR LES ÉQUIPEMENTS

```
Garage      : "garage pouvant accueillir [X] véhicules"
Terrasse    : "terrasse [ORIENTATION] idéale pour les soirées d'été"
Jardin      : "jardin arboré de [X]m², véritable havre de paix"
Cave        : "cave sèche de [X]m² pour vos aménagements"
Cave à vin  : "cave à vin climatisée — l'atout des amateurs"
Piscine     : "piscine [DIMENSIONS] chauffée pour profiter dès avril"
Cheminée    : "cheminée à bois, cœur battant des soirées d'hiver"
Poêle gran  : "poêle à granulés pour un chauffage économique et écologique"
DPE A/B     : "performance énergétique [CLASSE] — économies garanties"
Triple vit  : "triple vitrage pour un silence et une isolation parfaits"
Sous-sol    : "sous-sol aménageable de [X]m² — possibilités infinies"
Dressing    : "dressing sur mesure pour une organisation optimale"
Chambre par : "suite parentale avec espace privatif et salle d'eau attenante"
```

## EXEMPLES D'ANNONCES RÉUSSIES

### Maison 170m² — Saint-Ail
```
Bienvenue dans cette demeure d'exception où élégance et sérénité se conjuguent
à la perfection. Nichée sur un terrain de 2700m², cette maison de 170m² offre
à ses heureux propriétaires un cadre de vie privilégié, à l'abri des regards,
avec une exposition sud-est qui inonde chaque pièce de lumière naturelle.

L'architecture intérieure révèle des volumes impressionnants où les hauteurs sous
plafond généreuses accentuent la sensation d'espace. L'agencement ouvert favorise
une circulation naturelle entre les zones de vie, créant cette harmonie qui
transforme une maison en vrai foyer de famille.

La cuisine gastronomique avec son îlot central invite aux échanges et à la
convivialité. Le chauffage individuel au gaz assure un confort thermique optimal,
tandis que le classement DPE B témoigne de la qualité de l'isolation.

Cette propriété d'exception mérite d'être découverte — contactez notre équipe
pour une visite privée à votre convenance.
```
